# LineBot-Azure

## 一、簡介

此為我在Tibame的Azure課程繳交的作業，授課老師為陳奎銘。

此專案主要透過LineBot使用Azure的人臉與電腦視覺服務，當使用者上傳照片時，程式會依據圖片性質回傳Azure AI圖形辨識處理結果。

主程式為main.py，LineBot部署於Heroku。
